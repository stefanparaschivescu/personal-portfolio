Stock Checker

StockCheck is using Marketstack API to get the data and real-time updates of the stock market. I am still working on this project. The application programmable interface used doesn’t offer markets list, so I had to add my own database.

Users can find basic details about markets and stocks. The website is fully functional, altough the API doesn't deliver all the time. In the future I will use Alpha Vantage as it offers more in depth data, company overviews from experts and has a built-in Search Endpoint. This resolves my problem with the markets list and it’s afferent stocks. Also, beside raw data I want to implement charts to show the evolution of prices.

Features
  • Simple Menu with sections
  • Search option with smart Autocomplete from jQuery UI depending on the market selected
  • List of the markets to see it's stocks
  • Section with the best performed stocks from today
  • Stock prices and volume of the selected stock from the last 100 days

TodoList using Angular

Form submit doesn’t generate todo.id.
This web app uses a test API (not a real one) and so in the real world, the database would auto-increment the value of the id.

The project uses a service that runs all the features and simulates "get" and "post" methods using JSON Placeholder.

Because it doesn’t have a backend service to update the database, when the user deletes one of the todos it deletes every todos inserted. In future will have a small backend implementation and database with users to fix this bug.

Features
  • Add Todo
  • Cut Todo
  • Delete Todo
  • Information About Page
